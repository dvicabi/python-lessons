### שאלה 16

8 נקודות
כתוב קוד שקולט מספר דו ספרתי ומדפיס את ספרת העשרות שלו.

רמז: אפשר להשתמש ב־`//`.

## פתרון

```python
a = 9
b = 4
c = 2

a += b * c      # a = 9 + 8 = 17
b -= a % c + b // 2
# a % c = 17 % 2 = 1
# b // 2 = 4 // 2 = 2
# b = 4 - (1 + 2) = 1

c += a // b + c % 2
# a // b = 17 // 1 = 17
# c % 2 = 2 % 2 = 0
# c = 2 + 17 + 0 = 19

print(a, b, c)
```

התוצאה:

```python
17 1 19
```
