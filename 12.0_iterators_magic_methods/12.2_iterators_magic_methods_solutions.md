## 🧪 פתרונות לתרגילים

### תרגיל 1 – RangePrinter
```python
class RangePrinter:
    def __init__(self, start, end):
        self.current = start
        self.end = end

    def __iter__(self):
        return self

    def __next__(self):
        if self.current > self.end:
            raise StopIteration
        val = self.current
        self.current += 1
        return val

for number in RangePrinter(1, 5):
    print(number)
