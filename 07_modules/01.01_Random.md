# 🎲 ספריית `random` – ללמוד לשחק עם מזל 😄

## ✨ הסבר

לפעמים אנחנו רוצים שהמחשב יפתיע אותנו –  
יבחר מספר, שם או קלף **באקראי**! 🎯  

בשביל זה יש את **ספריית `random`** –  
היא מלמדת את פייתון "להגריל" דברים.

---

## 🔍 הסברים ודוגמאות

### קודם כול – צריך לייבא את הספרייה
```python
import random
````

כמו לומר למחשב: "תביא את הקופסה של הדברים האקראיים!"

---

### 🎯 בחירת מספר אקראי

```python
number = random.randint(1, 10)
print(number)
```

🪄 פייתון יבחר לך מספר **בין 1 ל־10** (כולל שניהם).
כל פעם שתריץ – תקבל תוצאה אחרת!

---

### 🎲 מספר עשרוני אקראי

```python
x = random.random()
print(x)
```

פייתון יבחר מספר **בין 0 ל־1**
(למשל 0.3726 או 0.91). זה טוב למשחקים או הדמיות.

---

### 🧍‍♀️ בחירת שם מרשימה

```python
names = ["Dana", "Avi", "Noa", "Lior"]
winner = random.choice(names)
print("המנצח הוא:", winner)
```

כל פעם המחשב בוחר מישהו אחר — מושלם להגרלות! 🎁

---

### 🔁 ערבוב רשימה

```python
cards = ["❤️", "♠️", "♦️", "♣️"]
random.shuffle(cards)
print(cards)
```

הרשימה מתערבבת כמו חפיסת קלפים! 🃏
עכשיו כל פעם הסדר אחר.

---

### 🎟️ בחירה של כמה פריטים

```python
numbers = [1, 2, 3, 4, 5, 6]
lotto = random.sample(numbers, 3)
print(lotto)
```

פייתון בוחר 3 מספרים שונים מהרשימה (בלי כפילויות).
למשל: `[2, 5, 6]` 🎲

---

### 🌱 סוד הקסם – `seed`

```python
random.seed(42)
print(random.randint(1, 10))
```

אם משתמשים ב־`seed`,
כל פעם שנריץ את הקוד – נקבל **אותה תוצאה בדיוק**!
זה שימושי כשרוצים תוצאות שחוזרות לצורכי ניסוי.

---

## 💡 טיפים קטנים

* `randint(1, 10)` כולל גם את 1 וגם את 10.
* `shuffle()` משנה את הרשימה עצמה, לא מעתיקה אותה.
* בשביל דברים חשובים באמת (כמו סיסמאות) – משתמשים בספרייה אחרת שנקראת `secrets`.

---

## ✅ בדוק את עצמך

1. איך גורמים למחשב לבחור מספר אקראי בין 1 ל־10?
2. מה ההבדל בין `choice()` ל־`sample()`?
3. מה עושה `shuffle()`?
4. למה כדאי לפעמים להשתמש ב־`seed()`?

---

## 🧩 תרגילים חווייתיים

1. כתוב קוד שמדפיס שם תלמיד אקראי מתוך רשימה של חברים בכיתה.
2. צור סימולציה של זריקת קובייה (מספר בין 1 ל־6).
3. ערבב רשימה של פירות והדפס את הסדר החדש.
4. צור משחק קטן שמבקש מהמחשב לבחור מספר בין 1 ל־5,
   ותבדוק אם המשתמש ניחש נכון! 😉
