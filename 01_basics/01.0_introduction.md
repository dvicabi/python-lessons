# 📘 מבוא לפייתון ותחביר בסיסי

## ✨ הסבר
תחשבו על פייתון כמו על שפה חדשה, אבל במקום לדבר עם אנשים, אתם מדברים עם המחשב.

במקום לומר "תדליק את האור", אתם כותבים שורות של קוד והמחשב עושה בדיוק מה שביקשתם.

פייתון היא שפה קלה, ידידותית, ואפשר לעשות איתה כמעט הכל, ממש כמו לגו לעולם התכנות!

כדי שהמחשב יבין אותנו, אנחנו צריכים לכתוב לפי כללי התחביר של פייתון.  
תחביר זה כמו כללי המשחק: אם שומרים עליהם, הקוד רץ חלק. אם לא, מקבלים שגיאה מהמחשב.

## 🔍 הסברים ודוגמאות

### מה זה פייתון?
פייתון היא שפת תכנות שנוצרה כדי להיות פשוטה להבנה, עם תחביר שנראה כמעט כמו אנגלית.

### למה ללמוד דווקא פייתון?
- היא קלה ללמידה, התחביר ברור.
- מתאימה לכל תחום: בניית אתרים, בינה מלאכותית, אוטומציות, משחקים ועוד.
- יש לה קהילה ענקית עם אינסוף מדריכים וחומרי לימוד.

### שורת הקוד הראשונה: Hello World
```python
print("Hello, world!")
````

### הסבר

* `print()` זו פונקציה שמדפיסה למסך.
* כל מה שנמצא בתוך הגרשיים יופיע במסך.

### דוגמה נוספת

```python
print("My name is Dvir.")
print("I love Python!")
```

### ריצת קוד

בפייתון הקוד מתחיל לרוץ מהשורה העליונה ביותר וממשיך שורה אחר שורה כלפי מטה.

הקוד נעצר באחד משני מצבים:

1. הגענו לשורה האחרונה, כלומר כל הקוד עבד.
2. נעצרנו בשגיאה, כלומר הקוד הפסיק בשורה שיש בה שגיאה ולא ימשיך לשאר שורות הקוד הבאות.

### מה זה תחביר בפייתון?

תחביר הוא אוסף החוקים שלפיהם כותבים קוד.

לדוגמה, בפייתון לא מספיק לכתוב למחשב מה לעשות, צריך לכתוב את זה בצורה נכונה:

```python
print("Welcome to Python!")
```

אם נשכח גרשיים, סוגריים או נכתוב בצורה לא תקינה, המחשב לא יבין אותנו ויציג שגיאה.

### תגובה: הערות בקוד

תגובה היא שורה בתוך קוד הפייתון שלא רצה כחלק מהקוד.

היא מתחילה תמיד בסולמית:

```python
# This is a comment
```

### למה משתמשים בתגובות?

לתגובה יש 3 שימושים מרכזיים:

1. להסביר בקצרה מה הקוד עושה.
2. לרשום משימות להמשך.
3. לבטל זמנית שורות קוד שרשמנו, כדי להשתמש בהן בעתיד.

### דוגמה לשימוש בתגובה

```python
# This is a comment
print("This will be printed")
```

בדוגמה הזאת:

* השורה שמתחילה ב־`#` היא תגובה.
* המחשב מתעלם ממנה.
* רק השורה עם `print()` באמת תרוץ ותוצג למסך.
