# 📘 המרות בין סוגי נתונים – Type Casting

## ✨ הסבר

המרה היא פעולה שבה אנחנו משנים את סוג הנתון של ערך מסוים
בפייתון זה קורה הרבה כי לא תמיד סוג הנתון שיש לנו מתאים לפעולה שאנחנו רוצים לבצע

לדוגמה
קלט מהמשתמש תמיד מגיע כמחרוזת
אבל אם נרצה לחשב איתו נצטרך להמיר אותו למספר

את ההמרות מבצעים באמצעות פונקציות מובנות
`str()` `int()` `float()` `bool()`

---

## 🔄 המרות לפי יעד ההמרה

### 🔤 המרה ל-str

כל סוג ניתן להמרה למחרוזת ותמיד מצליח

```python
print(str("hello"))
print(str(10))
print(str(3.14))
print(str(True))
print(str(False))
```

---

### 🔢 המרה ל-int

המרה למספר שלם
שימו לב לאילו מקרים מצליחים ואילו לא

```python
print(int("123"))
# print(int("12a"))      # ValueError מחרוזת לא מספר חוקי

print(int(7))

print(int(3.9))          # חיתוך החלק העשרוני
print(int(-3.9))

print(int(True))         # 1
print(int(False))        # 0
```

---

### 🔢 המרה ל-float

המרה למספר עשרוני

```python
print(float("3.14"))
# print(float("abc"))    # ValueError מחרוזת לא מספר חוקי

print(float(7))

print(float(2.5))

print(float(True))       # 1.0
print(float(False))      # 0.0
```

---

### ✅ המרה ל-bool

בדיקה האם הערך נחשב אמת או שקר

```python
print(bool("hello"))
print(bool(""))           # מחרוזת ריקה

print(bool(7))
print(bool(0))

print(bool(2.5))
print(bool(0.0))

print(bool(True))
print(bool(False))
```

---

## 🧠 כללי זהב להמרות

* `str` תמיד מצליח
* `int` ו־`float` ממחרוזת עובדים רק אם זה מספר חוקי
* המרה מ־`float` ל־`int` חותכת ולא מעגלת
* `bool`

  * ריק או אפס → False
  * כל דבר אחר → True
