# 📘 משתנים – Variables

## ✨ הסבר

משתנה הוא כמו קופסה שיש לה שם, ואנחנו יכולים לשמור בתוכה ערך

לתוך משתנה ניתן לשמור:

מספר

מילה (טקסט)

או כל סוג נתונים אחר

בכל פעם שנרצה להשתמש בערך השמור במשתנה, נשתמש בשם המשתנה

כללים לשמות משתנים והסיבה לכל כלל

שם המשתנה חייב להיות באנגלית
הסיבה: שפות תכנות נכתבות באנגלית, ושימוש בעברית או בשפות אחרות עלול לגרום לשגיאות, חוסר תאימות, וקושי בקריאת הקוד על ידי מתכנתים אחרים

שם המשתנה נכתב בלי גרשיים
הסיבה: גרשיים מגדירים טקסט (מחרוזת), ואם נכתוב שם משתנה עם גרשיים, המחשב יתייחס אליו כטקסט ולא כמשתנה

מותר להשתמש בקו תחתון (_) במקום רווחים
הסיבה: רווחים אינם חוקיים בשמות משתנים, והקו התחתון מאפשר לקרוא שמות ארוכים בצורה ברורה

שמות משתנים חייבים להיות ייחודיים
הסיבה: אם לשני משתנים יש אותו שם, המחשב לא ידע לאיזה ערך להתייחס ותחול שגיאה או דריסה של ערך קודם

שפת התכנות רגישה לאותיות גדולות וקטנות
הסיבה: המחשב מתייחס לכל שם בדיוק כפי שהוא נכתב
לכן שמות המשתנים text ו־Text הם שני משתנים שונים לחלוטין
---
## 🔍 הסברים ודוגמאות

### יצירת משתנה עם טקסט
```python
user_name = "Dvir"
```

### יצירת משתנה עם מספר
```python
user_age = 24
```

### הדפסת משתנים
```python
print(user_name)
print(user_age)
```

## 💡 טיפים מתקדמים
- שמות משתנים באנגלית בלבד, בלי רווחים, ובלי להתחיל עם מספר  
- אפשר לשנות ערך של משתנה מתי שרוצים – זו כל המשמעות של משתנה  
- אין צורך להגדיר את סוג המשתנה מראש – פייתון מזהה לבד  

## ✅ בדוק את עצמך

2. מה החוקים לשים שם משתנה?  
3. שאלה קשה איפה שמות המשתנים נשמרים במחשב?
