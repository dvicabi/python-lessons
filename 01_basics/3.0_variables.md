# 📘 משתנים – Variables

## ✨ הסבר
משתנה הוא כמו קופסה שיש לה שם – ואנחנו יכולים לשים בתוכה משהו ערך.  

אפשר לשים שם, מילה, מספר, או כל דבר אחר. 

וכל פעם שנרצה להשתמש בערך נרשום את שם הקופסא

שם הקופסא יהיה מופרד בקו תחתון ובלי גרשיים, על מנת שלא נקבל שגיאה

שמות המשתנים חייבים להיות ייחודים על מנת שלא נקבל שגיאה

זאת אומרת ששם המשתנה text והמשתנה Text הם יהיו 2 משתנים שונים לגמרי

## 🔍 הסברים ודוגמאות

### יצירת משתנה עם טקסט
```python
user_name = "Dvir"
```

### יצירת משתנה עם מספר
```python
user_age = 24
```

### הדפסת משתנים
```python
print(user_name)
print(user_age)
```

## 💡 טיפים מתקדמים
- שמות משתנים באנגלית בלבד, בלי רווחים, ובלי להתחיל עם מספר  
- אפשר לשנות ערך של משתנה מתי שרוצים – זו כל המשמעות של משתנה  
- אין צורך להגדיר את סוג המשתנה מראש – פייתון מזהה לבד  

## ✅ בדוק את עצמך
1. מהו משתנה?  
2. מה החוקים לשים שם משתנה?  
3. שאלה קשה איפה שמות המשתנים נשמרים במחשב?
