# 📘 משתנים וסוגי נתונים - Variables & Data Types

## ✨ הסבר

משתנה הוא כמו קופסה שיש לה שם, ואנחנו יכולים לשמור בתוכה ערך.

לתוך משתנה ניתן לשמור ערכים מסוגים שונים, לדוגמה: מספר, טקסט, בוליאני או כל סוג נתונים אחר.

בכל פעם שנרצה להשתמש בערך השמור במשתנה, נשתמש בשם המשתנה.

לדוגמה:

```python
user_name = "Dvir"
user_age = 24

print(user_name)
print(user_age)
````

הפלט יהיה:

```text
Dvir
24
```

---

## 🔍 כללים לשמות משתנים

שם המשתנה חייב להיות:

* באנגלית
* בלי גרשיים
* אם יש כמה מילים, מפרידים בקו תחתון `_`
* ייחודי וברור
* לא מתחיל במספר
* בלי רווחים

דוגמאות לשמות משתנים תקינים:

```python
user_name = "Dvir"
user_age = 24
is_student = True
```

דוגמאות לשמות לא תקינים:

```python
user name = "Dvir"
1age = 24
"user_name" = "Dvir"
```

בנוסף, פייתון רגישה לאותיות גדולות וקטנות.

לכן:

```python
text = "hello"
Text = "world"
```

אלה שני משתנים שונים לחלוטין.

---

## 🔍 יצירת משתנים

### יצירת משתנה עם טקסט

```python
user_name = "Dvir"
```

### יצירת משתנה עם מספר שלם

```python
user_age = 24
```

### יצירת משתנה עם מספר עשרוני

```python
user_height = 1.75
```

### יצירת משתנה בוליאני

```python
is_student = True
```

## 🔍 סוגי נתונים - Data Types

לכל דבר שאנחנו שומרים בפייתון יש סוג.

כמו ש"מים" הם נוזל ו"אבן" היא מוצק, גם בפייתון לכל ערך יש סוג נתון.

בפייתון יש הרבה סוגים שונים, למשל:

* מספרים
* טקסט
* בוליאנים
* רשימות
* סטים
* מילונים
* טאפלים

בשיעור הזה נתמקד בסוגים הבסיסיים ביותר.

---

## 🔍 סוגי נתונים בסיסיים

### מספר שלם - Integer / int

מספר שלם הוא מספר בלי נקודה עשרונית.

```python
user_age = 25
```

דוגמאות:

```python
10
0
-5
100
```

---

### מספר עשרוני - Float

מספר עשרוני הוא מספר עם נקודה עשרונית.

```python
user_height = 1.75
```

דוגמאות:

```python
3.14
1.5
0.75
-2.8
```

---

### מחרוזת - String / str

מחרוזת היא טקסט.

כל דבר שנמצא בין זוג גרשיים נחשב טקסט.

```python
user_name = "Dvir"
```

גם אם כתוב מספר בתוך גרשיים, הוא עדיין נחשב טקסט:

```python
grade = "100"
```

---

### בוליאני - Boolean / bool

בוליאני הוא ערך של אמת או שקר.

```python
is_student = True
is_finished = False
```

יש רק שני ערכים בוליאניים:

```python
True
False
```

---

## 🔍 בדיקת סוג הנתון

כדי לבדוק מה סוג הנתון של משתנה, נשתמש בפקודה:

```python
type()
```

דוגמה:

```python
user_age = 25
user_height = 1.75
user_name = "Dvir"
is_student = True

print(type(user_age))
print(type(user_height))
print(type(user_name))
print(type(is_student))
```

הפלט יהיה:

```text
<class 'int'>
<class 'float'>
<class 'str'>
<class 'bool'>
```

---

## 🔍 למה חשוב לדעת את סוג הנתון?

אם נזהה נכון את הסוג של כל דבר, נוכל לדעת אילו פעולות אפשר לבצע עליו ואיך להשתמש בו בצורה נכונה.

לדוגמה:

```python
age = 24
print("Age: " + age)
```

הקוד הזה יגרום לשגיאה, כי אי אפשר לחבר טקסט ומספר ישירות עם `+`.

אפשר לפתור את זה כך:

```python
age = 24
print("Age:", age)
```

או כך:

```python
age = 24
print("Age: " + str(age))
```

או כך:

```python
age = 24
print(f"Age: {age}")
```

בחירה נכונה של סוג הנתון עוזרת למנוע שגיאות ומאפשרת לכתוב קוד ברור, מדויק וקריא יותר.

---

## 💡 טיפים

* אפשר לשנות ערך של משתנה מתי שרוצים, זו כל המשמעות של משתנה.
* אין צורך להגדיר את סוג המשתנה מראש, פייתון מזהה לבד.
* שם משתנה טוב צריך להסביר מה נשמר בתוכו.
* אם ערך נמצא בתוך גרשיים, הוא טקסט.
* אם רוצים לבדוק סוג נתון, משתמשים ב־`type()`.
* פייתון רגישה לאותיות גדולות וקטנות.
