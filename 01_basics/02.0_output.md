# 📘 פלט בפייתון: הדפסות, גרשיים ותווי Escape

## ✨ הסבר
פלט הוא כל דבר שהתוכנית מציגה לנו על המסך.

בפייתון אנחנו משתמשים בפקודה:

```python
print()
````

כדי להציג הודעות, תוצאות, מספרים, טקסטים או מידע למשתמש.

לדוגמה, אם תוכנה היא כמו מזגן, אז הפלט הוא מה שהמזגן מציג או עושה לנו בחזרה:

```text
המזגן נדלק
הטמפרטורה היא 22 מעלות
מצב הפעולה הוא קירור
```

ובפייתון:

```python
print("The air conditioner is on")
print("Temperature: 22")
print("Mode: Cooling")
```

### הלמה

אנחנו צריכים פלט כדי לדעת מה התוכנית עושה, כדי להציג מידע למשתמש וכדי לבדוק אם הקוד שלנו עובד כמו שצריך.

---

## 🔍 הדפסה בסיסית

כדי להדפיס טקסט, שמים אותו בתוך גרשיים:

```python
print("Hello")
```

הפלט יהיה:

```text
Hello
```

אפשר גם להדפיס כמה שורות:

```python
print("Hello")
print("Welcome to Python")
print("Today we learn print")
```

הפלט:

```text
Hello
Welcome to Python
Today we learn print
```

### חשוב להבין

כל `print()` מדפיס שורה חדשה.

### הלמה

ברגע שכל הדפסה יורדת שורה, קל יותר לסדר מידע בצורה ברורה ומובנת.

---

## 🔍 הדפסת מספרים

אפשר להדפיס גם מספרים בלי גרשיים:

```python
print(100)
print(25)
print(3.14)
```

הפלט:

```text
100
25
3.14
```

אבל אם נשים מספר בתוך גרשיים, הוא נחשב טקסט:

```python
print("100")
```

הפלט עדיין ייראה כך:

```text
100
```

אבל מבחינת פייתון זה טקסט ולא מספר.

### הלמה

פייתון מבדילה בין מספר אמיתי שאפשר לחשב איתו לבין טקסט שרק מוצג למסך.

---

## 🔍 הדפסת טקסט עם גרש יחיד או גרשיים כפולים

בפייתון אפשר לכתוב טקסט בשתי דרכים:

```python
print("Hello")
print('Hello')
```

שתי הדרכים תקינות, ובשתיהן הפלט יהיה:

```text
Hello
```

כלומר, אין הבדל משמעותי בין גרש יחיד לבין גרשיים כפולים כאשר מדובר בטקסט רגיל.

### הלמה

פייתון מאפשרת לנו לבחור את הסגנון שנוח לנו, אבל חשוב להיות עקביים ולא להתבלבל.

---

## ⚠️ מתי אפשר להתבלבל עם גרשיים?

לפעמים בתוך הטקסט עצמו יש גרש או גרשיים, ואז צריך לבחור נכון איך לעטוף את המשפט.

### דוגמה 1: יש גרש בתוך המשפט

```python
print("I'm learning Python")
```

זה תקין.

אבל זה לא תקין:

```python
print('I'm learning Python')
```

למה?

כי פייתון חושבת שהטקסט נגמר אחרי האות `I`.

### דוגמה 2: יש גרשיים בתוך המשפט

```python
print('He said "Hello"')
```

זה תקין.

אבל זה לא תקין:

```python
print("He said "Hello"")
```

כי פייתון חושבת שהטקסט נגמר לפני המילה `Hello`.

### כלל פשוט

| בתוך המשפט יש       | נעטוף את המשפט ב    |
| ------------------- | ------------------- |
| גרש יחיד `'`        | גרשיים כפולים `" "` |
| גרשיים כפולים `" "` | גרש יחיד `' '`      |

### הלמה

אנחנו בוחרים את סוג הגרשיים לפי מה שיש בתוך הטקסט, כדי שפייתון תדע איפה הטקסט מתחיל ואיפה הוא נגמר.

---

## 🔍 הדפסה עם כמה ערכים יחד

אפשר להדפיס כמה דברים באותה שורת `print()` בעזרת פסיקים:

```python
name = "Dvir"
grade = 100

print("Student:", name)
print("Grade:", grade)
```

הפלט:

```text
Student: Dvir
Grade: 100
```

אפשר גם באותה שורה:

```python
print("Student:", name, "Grade:", grade)
```

הפלט:

```text
Student: Dvir Grade: 100
```

### הלמה

הפסיק בתוך `print()` מאפשר לנו לשלב טקסטים, מספרים ומשתנים בלי להסתבך בהמרות.

---

## 🔍 חיבור טקסטים עם `+`

אפשר לחבר טקסטים בעזרת הסימן `+`:

```python
first_name = "Dvir"
last_name = "Cabessa"

print(first_name + " " + last_name)
```

הפלט:

```text
Dvir Cabessa
```

אבל אי אפשר לחבר טקסט ומספר ישירות:

```python
age = 25
print("Age: " + age)
```

זו שגיאה.

הפתרון:

```python
age = 25
print("Age: " + str(age))
```

או פשוט יותר:

```python
age = 25
print("Age:", age)
```

### הלמה

הסימן `+` דורש ששני הדברים יהיו מאותו סוג. לכן בטקסט ומספר עדיף למתחילים להשתמש בפסיקים.

---

## 🔍 f-string: הדרך הנוחה לשלב משתנים בטקסט

f-string מאפשרת להכניס משתנים ישירות לתוך משפט.

```python
name = "Dvir"
grade = 100

print(f"The student {name} got {grade}")
```

הפלט:

```text
The student Dvir got 100
```

דוגמה בעברית:

```python
name = "דביר"
subject = "Python"

print(f"המורה {name} מלמד {subject}")
```

הפלט:

```text
המורה דביר מלמד Python
```

### הלמה

f-string הופכת הדפסות למובנות, נקיות וקלות לקריאה, במיוחד כשיש כמה משתנים בתוך משפט אחד.

---

## 🔍 תווי Escape: סימנים מיוחדים בתוך טקסט

לפעמים אנחנו רוצים להכניס לתוך טקסט דברים מיוחדים, כמו ירידת שורה או רווח גדול.

בשביל זה משתמשים ב־Backslash:

```text
\
```

שימו לב: זה לא Slash רגיל `/`.

---

## 🔍 ירידת שורה: `\n`

```python
print("Hello\nPython")
```

הפלט:

```text
Hello
Python
```

במקום לכתוב:

```python
print("Hello")
print("Python")
```

אפשר לכתוב הכל בהדפסה אחת:

```python
print("Hello\nPython")
```

דוגמה מסודרת יותר:

```python
print("Name: Dvir\nSubject: Python\nGrade: 100")
```

הפלט:

```text
Name: Dvir
Subject: Python
Grade: 100
```

### הלמה

`\n` עוזר לנו לסדר פלט בכמה שורות בלי לכתוב הרבה פקודות `print()`.

---

## 🔍 טאב: `\t`

`\t` מוסיף רווח גדול, כמו לחיצה על מקש Tab.

```python
print("Name\tGrade")
print("Dvir\t100")
print("Noa\t95")
```

הפלט בערך ייראה כך:

```text
Name    Grade
Dvir    100
Noa     95
```

### הלמה

`\t` עוזר לסדר פלט בצורה של טבלה פשוטה.

---

## 🔍 הדפסת גרשיים בתוך טקסט

אפשר להשתמש ב־Backslash כדי להדפיס גרש או גרשיים בלי לשבור את המשפט.

### גרש יחיד בתוך טקסט

```python
print('I\'m learning Python')
```

הפלט:

```text
I'm learning Python
```

### גרשיים כפולים בתוך טקסט

```python
print("He said \"Hello\"")
```

הפלט:

```text
He said "Hello"
```

### הלמה

ה־Backslash אומר לפייתון: "אל תתייחסי לסימן הבא כסיום הטקסט, אלא כחלק מהטקסט".

---

## 🔍 הדפסת Backslash עצמו: `\\`

אם רוצים להדפיס את הסימן `\`, צריך לכתוב אותו פעמיים:

```python
print("C:\\Users\\Dvir")
```

הפלט:

```text
C:\Users\Dvir
```

### טעות נפוצה

```python
print("C:\new_folder")
```

כאן פייתון עלולה להבין את `\n` כירידת שורה.

### הלמה

כי `\` הוא סימן מיוחד בפייתון. כדי להדפיס אותו כסימן רגיל, צריך לכתוב `\\`.

---

## 🔍 רשימת תווי Escape חשובים

| תו   | משמעות              | דוגמה              |
| ---- | ------------------- | ------------------ |
| `\n` | ירידת שורה          | `"Hello\nPython"`  |
| `\t` | טאב                 | `"Name\tGrade"`    |
| `\\` | הדפסת Backslash     | `"C:\\Users"`      |
| `\'` | הדפסת גרש יחיד      | `'I\'m here'`      |
| `\"` | הדפסת גרשיים כפולים | `"He said \"Hi\""` |

---

## ⚠️ טעות נפוצה: `/n` במקום `\n`

לא כותבים:

```python
print("Hello/nPython")
```

כי זה ידפיס ממש את הטקסט:

```text
Hello/nPython
```

הכתיבה הנכונה היא:

```python
print("Hello\nPython")
```

והפלט יהיה:

```text
Hello
Python
```

### הלמה

תווי Escape בפייתון מתחילים ב־Backslash `\`, לא ב־Slash רגיל `/`.

---

# 📘 דוגמה מאוחדת: פלט מסודר של תלמיד

```python
name = "Dvir"
subject = "Python"
grade = 100

print("Student Report")
print("--------------")
print(f"Name:\t{name}")
print(f"Subject:\t{subject}")
print(f"Grade:\t{grade}")
print("\nWell done!")
```

הפלט:

```text
Student Report
--------------
Name:       Dvir
Subject:    Python
Grade:      100

Well done!
```

בדוגמה הזאת השתמשנו ב:

* `print()`
* טקסט בתוך גרשיים
* משתנים
* f-string
* `\t`
* `\n`

### הלמה

זאת דוגמה שמראה איך בונים פלט אמיתי ומסודר, לא רק הדפסה בודדת.
