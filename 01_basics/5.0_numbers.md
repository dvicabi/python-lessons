# 📘 מספרים - Numbers

## ✨ הסבר
מספרים הם אחת מהשפות הכי פשוטות שהמחשב מבין.  
אם אנחנו רוצים לעשות חישובים כמו חיבור, חיסור, כפל או חילוק - פייתון תעשה את זה בשבילנו.  
רק צריך לכתוב את זה נכון, כמו במחשבון.`

## 🔍 הסברים ודוגמאות

### משתנים עם מספרים שלמים ועשרוניים
```python
a = 10        # int
b = 3.5       # float
```

### חיבור, חיסור, כפל, חילוק
```python
print(a + b)  # ועוד 13.5
print(a - b)  # פחות 6.5
print(a * b)  # כפול 35.0
print(a / b)  # חלקי 2.85
```

### חילוק שלם ושארית
```python
x = 5        # int
y = 2      # float

# כמה פעמים אתה יכול לחלק לכל אחד מהאנשים
print(x // y)  # 2

# מודולו - כמה שארית נשארה לך ביד
print(x % y) # 1

print(a // b)  # 2.0
print(a % b) # 3.0



```

### העלאה בחזקה
```python
print(2 ** 3)  # 8
```

## 💡 טיפים מתקדמים
- חילוק רגיל תמיד מחזיר float גם אם שני המספרים הם int  
- `//` זו דרך לבצע "חילוק עגול" בלי שארית  
- `**` זו הדרך להעלות בחזקה - לא להשתמש ב־^ כי זה משהו אחר בפייתון

## ✅ בדוק את עצמך

2. מה עושה `**` בפייתון?  
3. מה זה מודולו?
