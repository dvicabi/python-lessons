# 📘 מספרים ואופרטורים מתמטיים - Numbers & Arithmetic Operators

## ✨ הסבר

מספרים הם אחד מסוגי הנתונים הכי בסיסיים בפייתון.

אם אנחנו רוצים לעשות חישובים כמו חיבור, חיסור, כפל, חילוק, שארית או חזקה, פייתון יכולה לעשות את זה בשבילנו ממש כמו מחשבון.

אבל כדי שהמחשב יבין אותנו, צריך להשתמש בסימנים הנכונים.

הסימנים האלה נקראים **אופרטורים מתמטיים**.

לדוגמה:

```python
print(5 + 3)
````

הפלט יהיה:

```text
8
```

### הלמה

כמעט כל תוכנה אמיתית משתמשת במספרים: ציונים, מחירים, גילאים, נקודות במשחק, חישובים, ממוצעים, כמויות וזמנים. לכן חשוב לדעת לעבוד נכון עם מספרים ואופרטורים מתמטיים.

---

## 🔍 סוגי מספרים בפייתון

בפייתון יש שני סוגי מספרים בסיסיים שנשתמש בהם הרבה:

### מספר שלם - int

מספר שלם הוא מספר בלי נקודה עשרונית.

```python
age = 25
students = 30
score = 100
```

דוגמאות:

```python
10
0
-5
100
```

---

### מספר עשרוני - float

מספר עשרוני הוא מספר עם נקודה עשרונית.

```python
height = 1.75
price = 19.90
average = 87.5
```

דוגמאות:

```python
3.14
1.5
0.75
-2.8
```

---

## 🔍 משתנים עם מספרים

אפשר לשמור מספרים בתוך משתנים:

```python
a = 10
b = 3.5

print(a)
print(b)
```

הפלט:

```text
10
3.5
```

בדוגמה הזאת:

```python
a = 10
```

המשתנה `a` שומר מספר שלם מסוג `int`.

```python
b = 3.5
```

המשתנה `b` שומר מספר עשרוני מסוג `float`.

---

## 🔍 אופרטורים מתמטיים בסיסיים

```python
a = 10
b = 3
```

### חיבור

```python
print(a + b)
```

הפלט:

```text
13
```

---

### חיסור

```python
print(a - b)
```

הפלט:

```text
7
```

---

### כפל

```python
print(a * b)
```

הפלט:

```text
30
```

---

### חילוק רגיל

```python
print(a / b)
```

הפלט:

```text
3.3333333333333335
```

חשוב לדעת:
חילוק רגיל `/` תמיד מחזיר מספר עשרוני מסוג `float`, גם אם התוצאה נראית שלמה.

לדוגמה:

```python
print(10 / 2)
```

הפלט:

```text
5.0
```

---

## 🔍 חילוק שלם - //

חילוק שלם מחזיר כמה פעמים מספר אחד נכנס בתוך מספר אחר, בלי השארית העשרונית.

```python
x = 5
y = 2

print(x // y)
```

הפלט:

```text
2
```

למה?

כי 2 נכנס בתוך 5 פעמיים:

```text
2 + 2 = 4
```

ונשארת שארית של 1.

### דוגמה מחיי היום יום

אם יש לנו 5 סוכריות ו־2 ילדים:

```python
candies = 5
children = 2

print(candies // children)
```

הפלט:

```text
2
```

כל ילד יקבל 2 סוכריות שלמות.

### הלמה

משתמשים ב־`//` כשאנחנו רוצים לדעת כמה פעמים משהו נכנס בשלמות בתוך משהו אחר.

---

## 🔍 מודולו - שארית בחילוק - %

מודולו מחזיר את השארית שנשארת אחרי חילוק.

```python
x = 5
y = 2

print(x % y)
```

הפלט:

```text
1
```

למה?

כי 2 נכנס ב־5 פעמיים, וזה נותן 4.
אחרי זה נשארת שארית 1.

```text
5 = 2 + 2 + 1
```

### דוגמה מחיי היום יום

אם יש 5 סוכריות ו־2 ילדים:

```python
candies = 5
children = 2

print(candies % children)
```

הפלט:

```text
1
```

כל ילד יקבל 2 סוכריות, ותישאר סוכריה אחת בצד.

### הלמה

מודולו חשוב מאוד בתכנות. משתמשים בו כדי לבדוק שאריות, זוגי או אי זוגי, חלוקה לקבוצות, מחזוריות, תורות במשחקים ועוד.

---

## 🔍 העלאה בחזקה - **

בפייתון מעלים בחזקה בעזרת:

```python
**
```

לדוגמה:

```python
print(2 ** 3)
```

הפלט:

```text
8
```

כי:

```text
2 * 2 * 2 = 8
```

דוגמה נוספת:

```python
print(5 ** 2)
```

הפלט:

```text
25
```

כי:

```text
5 * 5 = 25
```

### חשוב מאוד

בפייתון לא משתמשים ב־`^` בשביל חזקה.

לא נכון:

```python
print(2 ^ 3)
```

נכון:

```python
print(2 ** 3)
```

### הלמה

במתמטיקה רגילה לפעמים כותבים חזקה עם סימן אחר, אבל בפייתון הסימן הנכון לחזקה הוא `**`.

---

## 🔍 דוגמה מאוחדת

```python
a = 10
b = 3

print("a + b =", a + b)
print("a - b =", a - b)
print("a * b =", a * b)
print("a / b =", a / b)
print("a // b =", a // b)
print("a % b =", a % b)
print("a ** b =", a ** b)
```

הפלט:

```text
a + b = 13
a - b = 7
a * b = 30
a / b = 3.3333333333333335
a // b = 3
a % b = 1
a ** b = 1000
```

בדוגמה הזאת השתמשנו בכל האופרטורים המתמטיים החשובים:

| אופרטור | משמעות       |
| ------- | ------------ |
| `+`     | חיבור        |
| `-`     | חיסור        |
| `*`     | כפל          |
| `/`     | חילוק רגיל   |
| `//`    | חילוק שלם    |
| `%`     | שארית בחילוק |
| `**`    | חזקה         |

---

## 🔍 ההבדל בין `/`, `//`, `%`

ניקח את הדוגמה:

```python
x = 5
y = 2
```

### חילוק רגיל

```python
print(x / y)
```

הפלט:

```text
2.5
```

כלומר: התוצאה המדויקת של החילוק.

---

### חילוק שלם

```python
print(x // y)
```

הפלט:

```text
2
```

כלומר: כמה פעמים 2 נכנס בשלמות בתוך 5.

---

### מודולו

```python
print(x % y)
```

הפלט:

```text
1
```

כלומר: כמה נשאר אחרי החילוק.

---

## 🔍 דוגמה מהחיים: חלוקת תלמידים לקבוצות

נניח שיש 29 תלמידים ורוצים לחלק אותם לקבוצות של 4.

```python
students = 29
group_size = 4

full_groups = students // group_size
left_over = students % group_size

print("Full groups:", full_groups)
print("Students left over:", left_over)
```

הפלט:

```text
Full groups: 7
Students left over: 1
```

המשמעות:

* אפשר ליצור 7 קבוצות מלאות
* יישאר תלמיד אחד ללא קבוצה מלאה

### הלמה

זו דוגמה שמראה איך חילוק שלם ומודולו עוזרים לפתור בעיות אמיתיות, לא רק תרגילים מתמטיים.

---

## 🔍 סדר פעולות חשבון בפייתון

פייתון שומרת על סדר פעולות חשבון כמו במתמטיקה.

קודם חזקה, אחר כך כפל וחילוק, ואז חיבור וחיסור.

```python
print(2 + 3 * 4)
```

הפלט:

```text
14
```

כי קודם מתבצע:

```text
3 * 4 = 12
```

ואז:

```text
2 + 12 = 14
```

אם נרצה לשנות את הסדר, נשתמש בסוגריים:

```python
print((2 + 3) * 4)
```

הפלט:

```text
20
```

כי קודם מתבצע:

```text
2 + 3 = 5
```

ואז:

```text
5 * 4 = 20
```

### הלמה

סוגריים עוזרים לנו לשלוט על סדר החישוב ולמנוע טעויות.

---

## ⚠️ טעויות נפוצות

### טעות 1: להשתמש ב־^ במקום **

לא נכון:

```python
print(2 ^ 3)
```

נכון:

```python
print(2 ** 3)
```

---

### טעות 2: לחשוב ש־/ מחזיר int

```python
print(10 / 2)
```

הפלט:

```text
5.0
```

גם כשהתוצאה שלמה, חילוק רגיל מחזיר `float`.

---

### טעות 3: להתבלבל בין // לבין %

```python
print(5 // 2)
print(5 % 2)
```

הפלט:

```text
2
1
```

`//` אומר כמה פעמים אפשר לחלק בשלמות.
`%` אומר כמה נשאר.

---

## 💡 טיפים מתקדמים

* חילוק רגיל `/` תמיד מחזיר `float`.
* חילוק שלם `//` מחזיר את החלק השלם של החילוק.
* מודולו `%` מחזיר את השארית.
* חזקה בפייתון עושים עם `**`.
* לא משתמשים ב־`^` בשביל חזקה בפייתון.
* כשיש חישוב מורכב, עדיף להשתמש בסוגריים כדי שיהיה ברור מה קורה קודם.

## 🧠 סיכום השיעור

בשיעור הזה למדנו לעבוד עם מספרים בפייתון.

למדנו שיש מספרים שלמים מסוג `int` ומספרים עשרוניים מסוג `float`.

בנוסף למדנו להשתמש באופרטורים מתמטיים:

* `+` לחיבור
* `-` לחיסור
* `*` לכפל
* `/` לחילוק רגיל
* `//` לחילוק שלם
* `%` לשארית בחילוק
* `**` לחזקה

הבנו גם את ההבדל בין חילוק רגיל, חילוק שלם ומודולו, וראינו איך הם עוזרים לנו לפתור בעיות אמיתיות כמו חלוקה לקבוצות.

### הלמה הגדול של השיעור

מספרים ואופרטורים מתמטיים הם הבסיס לכל חישוב בתוכנה.

ברגע שאנחנו יודעים לבצע חישובים בצורה נכונה, אנחנו יכולים לבנות תוכנות שמחשבות ציונים, מחירים, ממוצעים, זמנים, נקודות במשחקים ועוד הרבה דברים אמיתיים.
